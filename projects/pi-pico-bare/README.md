# Pi-Pico-Bare

This is a sample project for the [Raspberry Pi Pico](https://www.raspberrypi.com/documentation/microcontrollers/pico-series.html#pico1) board, which uses the [RP2040](https://www.raspberrypi.com/documentation/microcontrollers/microcontroller-chips.html#rp2040) MCU.  The MCU provides two 125MHz Cortex M0+ cores, no on-chip flash, and 264KB of RAM in a 56-pin QFN packate.  The Pico board integrates a 2MB QSPI flash chip.

This sample project is a "bare-metal" implementation.
