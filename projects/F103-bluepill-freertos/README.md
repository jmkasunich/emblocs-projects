# F103-BluePill-FreeRTOS

This is a sample project for the common "blue pill" board containing an [STM32F103C8T6](https://www.st.com/en/microcontrollers-microprocessors/stm32f103c8.html) chip from ST-Micro.  This CPU has a 72MHz Cortex M3 core, 64KB flash, and 20KB RAM in a 48-pin LQFP package.

Data from STMicro includes:
[Datasheet](https://www.st.com/resource/en/datasheet/stm32f103c8.pdf)
[Reference Manual](https://www.st.com/resource/en/reference_manual/rm0008-stm32f101xx-stm32f102xx-stm32f103xx-stm32f105xx-and-stm32f107xx-advanced-armbased-32bit-mcus-stmicroelectronics.pdf)
[Programming Manual](https://www.st.com/resource/en/programming_manual/pm0056-stm32f10xxx20xxx21xxxl1xxxx-cortexm3-programming-manual-stmicroelectronics.pdf)
[many application notes](https://www.st.com/en/microcontrollers-microprocessors/stm32f103c8.html#documentation)

This sample project uses FreeRTOS for its real-time threads.
