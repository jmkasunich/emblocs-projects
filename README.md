# emblocs-projects

This repo contains some sample projects that use the emblocs library.
It is the result of a refactor of emblocs; the original emblocs
contained some sample projects and a build system, but I decided
that I want emblocs to be more of a code-only library.  I'm building
projects on multiple different platforms with different build systems,
so I am going to remove the build system from the original repo.

This repo will include emblocs as a sub-module.

At this time, most of the sample projects are empty.  They represent goals, not working code.
